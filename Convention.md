# 🖋 작성 규칙
코드 작성하실 때 규칙을 따라주세요.
<br/><br/>
## 📌 네이밍
- 변수명 : camelCase (예: appleJuice)
- 클래스명 : PascalCase (예: AppleJuice)
- 메서드명 : camelCase (예: appleJuice())
- 상수명 : 모두 대문자 + 언더스코어 (예: APPLE_JUICE)
- 컬럼명 : 모두 대문자 + 언더 스코어
<br/><br/>
## 📌 주석 및 문서화 규칙
- 주석 언어 : 한국어
<br/><br/>
## 📌 커밋 메시지
- 수정/추가 사항 간단하게 기재
